四、配置跨域 #

修改配置文件，在配置文件中新增server对象，对host、port、proxy进行配置。

export default defineConfig({
  plugins: [vue()],
  server:{
    port:3100,
    host:"0.0.0.0",
    proxy:{
      '/login': 'http://localhost:5000',
    }
  }
})
