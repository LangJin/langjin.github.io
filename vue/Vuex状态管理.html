状态管理 #

vue的状态就类似于flask中的全局变量。 在vue项目中的不同的页面中，可能会采用同一个状态来控制组件的变化，比如登录后，导航栏从未登录显示为用户头像这种场景。就需要用到状态管理来进行控制。

Vue3自带的实现方式 #

在vue中一般使用store来作为全局的状态的名字，因为store是在全局项目中使用的，所以我们必须把store挂在到我们创建的应用中。 所以这种写法，只能写到APP.vue中，或者在其他地方写了后在APP.vue中导入。

  • 创建全局状态
    JavaScript
    <script>
import {reactive} from 'vue'
const store = {
    state:reactive({
        message:"HelloWorld!"
    }),
    setMassage(value){
        this.state.message = value;
    }
}
export default {
    provide:{  //利用provide把store提供给其他组件使用
        store
    }
}
</script>

Vuex的关键字说明 #

  • State，用于定义变量
  • Getter，用于对State进行筛选
  • Mutation，用于操作修改State
  • Action，用于操作函数方法。
  • Module，定义不同的模块，可以把状态分成不同的模块。

创建状态管理 #

  • 创建store文件 在项目中创建store文件夹，在文件夹的index.js文件中写入以下内容：
    JavaScript
    import { createStore } from 'vuex'
export default createStore({
    state: {
        count: 0
    },
    mutations: {
    // 进行数据更新，改变数据状态
        countType(state, num){
            state.count =  state.count + num
        }
    },
    actions: {
    //执行动作，数据请求
        addCount({commit}){
            fetch('../data.json')
            .then(function(response) {
                return response.json();
            })
            .then(function(myJson) {
                let action = {
                    type: 'countType',
                    payload: myJson.text
                }
                commit(action)
            });
        }
    },
    getters: {
    // 获取到最终的数据结果
        getCount(state){
            return state.count
        }
    }
})

  • 在组件中使用
    JavaScript
    import { reactive,onMounted } from 'vue'
import { useStore } from 'vuex'
export default {
    setup(){
        let store = useStore()
        let data = reactive({
            count: 0
        })
        onMounted(() => {  //监听数据变化
            store.dispatch('addCount')
        })
        let test = () => {
            store.commit('countType',3)  //commit操作mutations
        }
        let add = () => {
            store.dispatch('addCount')  //dispatch使用action方法
            data.count = data.count = store.getters.getCount
        }
        return{
            data,
            add
        }
    }
}
