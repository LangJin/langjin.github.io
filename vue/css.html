CSS就是层叠样式表，它的作用是用来实现网页的UI效果的。比如翻页的动化效果，鼠标点击或者移动的动态效果，UI显示的颜色图片之类的效果，都是用CSS来实现的。 CSS现在流行的是CSS3的版本。

快速上手 #

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        h1{
            color: aqua;
        }
        #demo{
            font-size: 200px;
        }
        .demo1{
            text-align: center;
        }
    </style>
</head>
<body>
    <h1 id="demo" class="demo1">测试</h1>
</body>
</html>

选择器 #

css的选择器一共有三种，分别就是标签，id，class选择器。

  • 标签选择器 标签选择器里面写的样式，会影响所有名字是这个标签的样式。
css
h1{
    color: aqua;
}

  • id选择器 id选择器需要使用#来定义。比如：#yangtai
css

    color: aqua;

style属性 #

样式除了写在style标签里面，还可以写到标签的style属性中。 比如：

html
 <div style="font-size: 30px;background-color: coral;">客厅</div>
1

css文件的导入 #

当一个网页的样式很多的时候，全部都写到style标签中十分难以管理和维护。 这种情况，我们就可以把css样式写到一个css文件中，然后再在head标签中，使用link标签导入到网页中。

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="./test.css">
    <title>Document</title>
</head>
<body>
    <h1 id="demo" class="demo1">测试</h1>
</body>
</html>
