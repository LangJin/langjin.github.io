JavaScript这个脚本语言是运行在浏览器中的。 JavaScript可以为网页提供丰富的交互的功能，比如数据的处理，接口的调用，页面的跳转等等，这些都是由JavaScript来实现的。 并且JavaScript还可以操作html、css的内容。对他们的内容进行新增或者修改之类的操作。 JavaScript还可以去操作浏览器提供的功能，比如cookies。

快速上手 #

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script>
        function test(){
            console.log("你怎么才点我啊")
        }
    </script>
    <title>浪晋的主页</title>
</head>
<body>
    <button onclick="test()">点我</button>
</body>
</html>

JavaScript文件的导入 #

当一个页面用到JavaScript的内容很多的时候，或者说一个方法在多个网页中都有用到，我们就可以把JavaScript的脚本抽离成一个单独的js文件，在网页中直接导入这个文件就可以用了。

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script type="text/javascript" src="./demo.js"></script>
    <title>浪晋的主页</title>
</head>
<body>
    <button onclick="test()">点我</button>
</body>
</html>

条件判断 #

注意：在js中，判断条件的逻辑连接，&&表示并且，||表示或者。 js的代码不是通过缩进来表示的代码块，缩进是无所谓，js是通过{}来实现的代码块的识别，所以{}务必要成对的出现。 每一句代码写完，标准写法是建议用;结尾的，但是不写也行。

const 方法名 = (参数1,参数2) =>{    //es6的语法糖的写法
    //具体的业务逻辑
    return 参数1;   //如果不返回数据，也可以不写return
    return; //会终止这个方法的运行。
