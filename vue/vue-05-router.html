Vue Router是Vue官方实现的一个路由管理插件。

安装 #

cmd
npm install -S vue-router@4
1

Vue Router的功能点 #

  • 创建路由
  • 嵌套路由
  • 监控路由
    • route
      1. route.params
      2. route.query
  • 操作路由
    • router
      1. router.push() 导航到某个路由
      2. router.replace() 替换当前路由为某路由
      3. router.go() 访问历史记录，参数可以写 1，2，3，-1，-2，-3等。
  • 路由守卫 router.beforeEach()
  • 动态路由
    • router.addRoute()：添加路由
    • router.removeRoute()：删除路由
    • router.hasRoute()：检查路由是否存在。
    • router.getRoutes()：获取一个包含所有路由记录的数组。

封装路由方法 #

在安装完成Vue Router后，需要在项目中创建一个路由文件。 一般是在src中创建一个router文件夹，然后在里面创建index.js的文件。 具体内容如下：

JavaScript
//首先从vue-router中导入所需要的方法
import {createRouter,createWebHashHistory,createWebHistory,beforeEach} from 'vue-router'
import Home from '../components/Home.vue'  //导入需要组件
import User from '../components/User.vue'  //导入需要组件
const routes = [  //创建访问组件的路由地址
    {
      path:"/",
      component:Home,
      meta: { requiresAuth: true } //路由的元信息，里面可以设置一些额外的信息
    },
    {
      path:"/user",  //路由地址
      name:"用户",  //路由名称
      alias:"/userinfo",  //alias是对路由取的别名，也就是说使用path和alias的效果是一样的。
      component:User,  //访问的组件
      beforeEnter:(to,from,next) =>{
        return flase  //独享的路由守卫
      }
    },
    {
      path:"/user/:id",  //id是一个参数，这是动态路由的写法
      alias:['/userinfo','users'],  //可以取多个别名
      redirct:to=>{      //重定向
        return {path:'/home',query:{name:"张三"}}   //这里面也可以根据to作为参数来写更复杂的逻辑，最后return返回最终的路由即可
      },  
      component:User
    },
    {
      path:"/user/:id",  //id是一个参数，这是动态路由的写法
      redirct:to=>{      //重定向
        return {path:'/home',params :{name:"张三"}}   
      },  
      component:{   //对视图进行命名，以方便在同一个路由加载，分别加载不同的组件。
        default:User,
        footer:Footer
      }
    },
]
const router = createRouter({
    history: createWebHashHistory(),  //使用hash模式 
    // history: createWebHistory(),  //使用传统模式 
    routes,   //加载路由地址
  })


router.beforeEach((to,from,next)=>{   //路由守卫，to表示即将要进入的目标的信息，from表示当前导航正要离开的路由。这个方法会在每次请求路由的时候调用
  //在中间就可以写上逻辑处理。一般用来处理用户是否访问某路由的权限。比如，是否登录。
  // next()  //表示允许继续
  // return flase  //表示取消请求
})

export default router;  //导出

    <router-view></router-view> 
    <router-view name="user"></router-view> 
  </div>
</template>
