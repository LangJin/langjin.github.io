vue的项目的创建 #

vite是一个前端的脚手架工具，所谓的脚手架的意思就是可以通过这个工具快速的生成一个项目的骨架。 并且vite也是一个前端的调试和编译工具。 利用以下命令就可以快速的生成一个前端的基础项目。

  • 创建项目
cmd
npm init vite
1
  • 安装项目所依赖的包 使用vscode打开了我们上一步创建的项目后，在终端中输入npm i完成对应的依赖的安装。
  • 启动项目
npm run dev
1

注意：vue做的项目是网页，网页是在浏览器中运行的，所以当我们启动了项目后，我们是需要在浏览器中去访问我们的项目的，才能看到具体的运行结果。并且我们是可以利用浏览器的开发者工具去调试检查我们的项目的。

vue的项目的结构 #

sh
node_modules        # 项目所需要的第三方的包
pubilc              # 项目所需要一些公共文件或者静态资源
src                 # 项目的核心的代码
    assets          # 项目的静态资源
    components      # 项目封装的组件
    App.vue         # 项目工程的入口文件
    main.js         # 项目所需要导入的第三方的包都得写到这里
    style.css       # 项目的公共样式的css文件
.gitignore          # git的配置文件，告诉git哪些文件可以忽略
index.html          # 项目的访问的入口文件
package-lock.json   # 我们本地安装的第三方的依赖的详细信息
package.json        # 项目所需要的第三方的包的信息，以及项目基本信息
README.md           # 项目的说明文件
vite.config.js      # vite的配置文件

快速上手 #

vue
<template>
  <div class="demo">浪晋的主页</div>
  <p>姓名：{{name}}</p>
  <p>年龄：{{age}}</p>
  <button @click="setAge()">点我+1岁</button>
</template>
<script >
export default{
  name:"",  // 给这个页面取个名字
  data(){   //对给template使用的变量进行定义
    return {
      name:"浪晋",
      age:23
    }
  },
  methods:{ //定义的方法就得写到这里
    setAge(){
      this.age++
    },
  }
}
</script>
<style>
.demo{
  color: brown;
  font-size: 60px;
}
</style>

template中模板语法的使用 #

vue的变量的定义要写到data里面。只有写在了data里面的变量，我们才可以在template里面显示出来。在template里面是通过的方式去访问我们的变量的。的这种写法叫做模板语法。

vue
<template>
  <div class="demo">浪晋的主页</div>
  <p>姓名：{{name}}</p>
  <p>姓名的长度：{{name.length}}</p>
  <p>年龄：{{age}}</p>
</template>
<script >
export default{
  name:"",  // 给这个页面取个名字
  data(){   //对给template使用的变量进行定义
    return {
      name:"浪晋",
      age:23
    }
  },
}
</script>

template中变量的读取 #

因为template里面写的代码本质上就是html的代码。所以我们html的各种标签的属性在vue中都是可以使用的。 所以我们可以把属性具体的内容写到data中，作为一个变量来给template用。

vue
<template>
  <div class="demo">浪晋的主页</div>
  <p>姓名：{{name}}</p>
  <p>年龄：{{age}}</p>
  <p>姓名的长度：{{name.length}}</p>
  <p>{{age > 12 && age < 18 ? '青少年':age >= 18?'成年':'未成年'}}</p>
  <button v-on:click="setAge()">点我+1岁</button>
  <button @click="setAge()">点我+1岁</button>
  <p v-text="demo" v-bind:class="class1"></p>
  <p v-text="demo" :class="class1"></p>
  <div v-html="html1"></div>
  <h1 v-show="status">你能不能看到我啊</h1>
  <p>{{value1}}</p>
  <input type="text" v-model="value1">
  <p>--------------------------------------------------</p>
  <p v-if="age > 12 && age < 18">青少年</p>
  <p v-else-if="age >= 18">成年</p>
  <p v-else>未成年 </p>
  <div>
    <p>这是循环出来的内容</p>
    <p v-for="value,index in alist">第{{index}}条数据:{{value}}</p>
  </div>
</template>
<script >
export default{
  name:"",  // 给这个页面取个名字
  data(){   //对给template使用的变量进行定义
    return {
      name:"浪晋哈哈哈",
      age:10,
      demo:"今天星期三",
      html1:"<h1>哈哈哈</h1>",
      status:true,
      value1:"邓子键茂密的头发",
      class1:"demo",
      alist:["张三",'李四','王五']
    }
  },
  methods:{ //定义的方法就得写到这里
    setAge(){
      this.age++
    },
  }
}
</script>
<style>
.demo{
  color: brown;
  font-size: 30px;
}
</style>

属性说明
v-text等同于html的text属性，作用是可以直接读取变量来作为文本展示在标签中，使用了v-text标签中就不能写其他的值，会冲突。
v-html读取变量，并且以html的内容来进行展示。使用了v-html标签中就不能写其他的值，会冲突。
v-show控制某个标签是否显示
v-model绑定数据的显示，双向绑定，一般用于输入框。
v-bind绑定属性的，比如id，class之类的属性。完整的写法是v-bind:属性="变量",缩写:属性="变量"
v-on绑定事件，比如点击事件。完整的写法v-on:click="方法",缩写@click='方法'
v-if用来做判断的，v-if,v-else-if,v-else这三种。
v-for用来做循环,<p v-for="value,index in alist" :key="index">第条数据:</p>,value是具体的值，index是下标，如果循环的是一个对象，index就是对应的key
