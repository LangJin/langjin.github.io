python的开发环境搭建 #

python是我们编程所需要的代码的运行环境，如果没有python的运行环境，我们的python代码是无法运行的。 pip是python的第一个第三方的包管理工具。我们在使用的时候，会用到很多别人已经写到的代码包，这些包的安装和管理就需要通过pip工具来进行。 所以在安装完成后，需要去检查python和pip是否能够成功的使用。

安装python #

1、下载安装包 https://www.python.org/downloads/release/python-3106/ 2、安装 下载完成后，直接双击就可以了。 注意勾选add python to path选项。

sh

Python 3.10.4
C:\Users\WDAGUtilityAccount>pip -V
pip 22.0.4 from C:\Program Files\Python310\lib\site-packages\pip (python 3.10)

如果在命令行中输入以上命令没有正确的输出的时候，需要自己去检查环境变量，配置的是否正确。

VScode #

python的环境搭建好了后，我们要去写代码，是需要用到代码的编辑器的。 python的代码的编辑器很多，比如VScode、pycharm等等。 在这里不需要去纠结到底选哪一个，你喜欢哪个编辑器就用哪个就行了。 不过在这里我推荐大家使用VScode，因为在后面我们写前端的代码的时候，也是用VScode，我们就不需要额外的安装更多的软件了。

下载 #

下载地址：https://code.visualstudio.com/

安装 #

直接双击下一步即可，不需要做任何的操作。

VScode的插件 #

VScode有很多各种各样的插件，我们就可以根据我们自己的需要去安装自己需要的插件。 常用的插件

  • chinese中文插件，可以让VScode页面变成中文显示
  • Atom One Dark Theme 代码的配色的主题
  • vscode-icons 文件图标的主题
  • python 我们如果要用vscode来写python的代码，我们就需要安装python的插件，这样子VSCODE才能识别你写的语法格式，才会给你做对应的提示
  • MySQL 这个插件支持mysql和redis的图形化管理，我们就可以直接使用VScode去操作数据库了。
  • Project Manager 这是用来管理你的代码的项目的插件，通过这个插件就可以实现在不同的项目中互相切换，
  • Git History 可以查看 git的提交记录
  • Office Viewer 可以直接在VSCODE上打开office的文件

搭建前端的运行环境 #

搭建nodejs的运行环境，现在的前端的网站的代码之类的，都是利用nodejs来写的。 nodejs等同于python这样的代码运行环境。 下载地址：https://nodejs.org/en/

检查是否安装成功 #

node -v 检查nodejs的运行环境是否准备好了 npm -v # npm就等同于python的pip工具，它就是前端的一个第三方的包管理工具

环境的配置 #

npm在下载第三方的包的时候，下载的内容很多，乱七八杂的。 所以我们最好去指定它下载到什么地方。 自己任选2个合适的文件夹，一个下载的文件的叫global，一个缓存的文件的叫cache。 创建好了文件夹后，我们需要把global配置环境变量中。 通过命令，告诉npm，对应的文件夹的位置。

npm config set prefix "global的路径" npm config set cache "cache的路径"

安装其他的第三方的包管理工具 #

cnpm yarn

创建一个前端的项目 #

npm init vite
1

npm i

npm run dev

【python3和pip3的安装】 1、安装各种需要的依赖包和组件，有两个命令。 yum install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gcc make yum install libffi-devel 2、下载python wget https://www.python.org/ftp/python/3.7.0/Python-3.7.0.tgz 3、解压安装，四个步骤。 tar -zxvf Python-3.7.0.tgz cd Python-3.7.0 ./configure make && make install 4、配置环境变量 mv /usr/bin/python /usr/bin/python.bak ln -s /usr/local/bin/python3 /usr/bin/python mv /usr/bin/pip /usr/bin/pip.bak ln -s /usr/local/bin/pip3 /usr/bin/pip 5、验证是否安装成功，两个命令。 python -V pip -V 6、配置yum，不然这个就不能用了。 vi /usr/libexec/urlgrabber-ext-down 把第一行的python改为python2.7 vi /usr/bin/yum 把第一行的python改为python2.7
