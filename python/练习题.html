"""
现在有一个数组A。[1,2,3,4,8,9,88]
你现在通过input输入一些数据，然后保存到一个数组B中，把数组B中的数据合并到数组A中，
但是如果数组B中的数据是A中已存在的数据，那么重复的数据就不合并，并且输出合并成功了多少条，重复的有多少条。哪些数据是重复的。

注意：不允许使用set
"""
a = [1,2,3,4,8,9,88]
b = []
for i in range(7):
    value = int(input("请输入整数："))
    b.append(value)
print(a)
errorList = []
okList = []
for i in a:
    for j in b:
        if i == j:
            errorList.append(j)
for i in b:
    if i not in errorList:
        okList.append(i)
a.extend(okList)
print("结果：",a)
print("合并成功了{}条".format(len(okList)))
print("失败了{}条，分别是{}".format(len(errorList),errorList))




"""
计算任何一个日期是当年的第几天
"""
year = int(input("请输入年份："))
month = int(input("请输入月份："))
day = int(input("请输入日期："))
monthlist = [31,28,31,30,31,30,31,31,30,31,30,31]
days = 0
if year % 4 == 0 and year % 100 != 0 or year % 400 == 0:
    monthlist[1] = 29

    for i in range(month-1):
        days = days + monthlist[i]
    days = days + day
else:
    for i in range(month-1):
        days = days + monthlist[i]
    days = days + day
print("今天是今年的第{}天".format(days))

一个男孩去电影院看电影。已知电影院会同时开始播放两部时长不一样的片子，每部片子放完就会从头播放。 男孩迟到了t分钟，请问怎么样才能知道他最快多少分钟可以看到新的片子。 第一部片子的时长是a分钟，第二部的时长是b分钟。

举例：小男孩知道第一部片子是3分钟，第二部片子是7分钟。他迟到了10分钟，第一部片子已经播放了三次，目前正在播放第四次，所以是：12-10=2，而第二部片子还在播放第二次：14-10=4，所以他最快2分钟可以看到新的片子。

a、b、t都是整数，且都是正数。

【程序要求】 输入： 3（第一个片子的时长） 7（第二个片子的时长） 10（迟到的时间）

输出： 2

a = int(input()) b = int(input()) t = int(input()) print(min(a - t % a, b - t % b))

现在互联网.上流行用缩写来表示一些词汇。比如dbq等于[对不起]，xswl等于[笑死我了], cxk等于 [蔡徐坤]等等。请构建一个黑话字典，用来将黑话还原成正常句子。 输入格式: [包含黑话的字符串,黑话]用英文逗号分隔 黑话字典: heihua_ dict =

py

hdict = {
    "xswl":"笑死我了",
    "cxk":"蔡徐坤",
    "xhz":"小黑子",
    "dbq":"对不起"
}
heihua = "xswl,你们都是xhz,dbq,cxk"
for i in hdict:
    if i in heihua:
        heihua = heihua.replace(i,hdict[i])
print(heihua)
