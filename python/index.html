安装python的编程环境 #

1、直接双击python的安装包 2、勾选 ADD python310 to PATH 3、选择自定义安装 4、勾选给所有人安装 5、全程默认下一步即可。 6、打开cmd，输入python -V和pip -V检查python是否安装成功即可。 python就是我们的python程序运行环境。 pip是python的第三方的包的管理工具。

安装vscode #

vscode是微软出品的一个免费的代码编辑工具，几乎现在大多数的编程语言的编写，他都是支持的。

python的语法基础 #

python的变量和赋值 #

比如：

py
a = "张三"
1

把张三这个值，赋值给了a这个变量， 在这里a就是一个变量。 等号的作用就是赋值 给变量赋值的数据是什么类型，那么这个变量也会是对应的类型。

python自带的方法 #

  • print() 作用就是打印输出括号里面的内容。

  • input() 作用就是让用户可以通过这个方法输入内容,input获取的数据是字符串类型的

  • type() 作用是可以识别数据的类型

  • int() 作用是把字符串类型的数据和float类型的数据，转行成整数类型！ 这个数据必须要长的像数字才可以转换！

  • float() 可以把字符串类型的数据和int类型的数据转换成浮点型的数据！ 不过字符串的数据，要求必须长得像阿拉伯数字才可以！

  • round() 用来处理小数问题的，可以用这个方法来指定显示多少位的小数，并且自动的四舍五入

  • list() 把数据转行成数组格式

  • range() 数列生成器

  • len() 获取对象的长度，适用于字符串、元组、数组、字段、集合

py
a = "今天你吃了吗"
print(len(a))  # 6

只要是用了引号引起来了的内容，都是字符串类型！如果不是字符串类型的数据，你就不要给他加引号。 比如："hahah" "123" "你好"

浮点型类型/float #

元组类型/tuple #

元组是python里面的数据的一个集合，元组里面的数据是不可以修改的。 比如：a = (1,2,3,"哈哈") 对于元组里面的数据的操作，需要用到下标去操作。 python会自动的给放到元组里面的数据编号。但是计算机编号是从0开始的。 这里的编号就是我们所谓的下标！ 比如我们现在要操作a这个元组里面的“哈哈”。a[3]

注意：python的下标和其他的编程语言不一样的是，可以同步负数倒着取。比如，最后一位的下标就是-1

  • index 获取某个值的下标，如果元组里面有多个相同的值，只会获取第一个出现的值的下标
py
a = (1,2,3,"哈哈")
b = a.index("哈哈")  # b就是哈哈这个值在a这个元组中的下标
print(b)

字典类型/dict #

在其他的编程语言里面把python字典叫做对象。 字典里面的数据是以键值对(key:value)的方式来存放的。 字段的格式和json的格式是一样的。 字典里面没有顺序的说法。

py
a = {"name":"张三","age":18}
b = a["name"]
print(b)

集合/set #

集合是python里面的一个无序不重复的数据组 比如：{1,2,3,4} 集合可以对重复的数据进行过滤，所以我们可以利用集合来去重 当我们要创建一个空的集合的时候，不能直接写{} 因为{}表示的一个空字典 所以我们要用set()

  • 两个集合相减，求差集 a = {"a","b","c"} b = {"a","e","d"} print(a - b)

  • 求两个集合的交集 a = {"a","b","c"} b = {"a","e","d"} print(a | b)

  • 求两个集合的并集 a = {"a","b","c"} b = {"a","e","d"} print(a & b)

  • 得出两个集合中没有重复的数据 a = {"a","b","c"} b = {"a","e","d"} print(a ^ b)

py

    """
    方法名字的必须是小写字母开头的。
    这里写对这个方法的说明。
    x，y是这个方法接收的参数
    """
    pass
