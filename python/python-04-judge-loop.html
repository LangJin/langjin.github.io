python的循环 #

当我们想要重复的运行一块代码的时候，我们就可以使用循环来实现。

  • while循环 while循环是通过while后面的条件进行的，只要满足条件就会一致循环下去。
py
while Ture:
    print("循环打印")

2

  • for循环 for循环是通过遍历的手段来实现的。

  • break 在循环的过程中如果遇到了break就结束这个循环

  • continue 在循环的过程中如果遇到了continue就跳过这一次循环

练习1 #

有一个数组，里面存放的都是学生的成绩， 现在要求把60分以上的成绩和60一下的成绩分别存放到两个数组中

py
a = [23,34,45,65,43,12,78,87,98,55,45,78,98,84,83,89]
highlist = []
lowlist = []
for i in a:
    if i >= 60:
        highlist.append(i)
    else:
        lowlist.append(i)
print(highlist)
print(lowlist)
