MySql是一个经典的关系型数据库，大多数公司的项目使用的数据库都是它。

安装 #

在Windows上安装mysql #

在Linux上安装MySQL #

  1. 通过xftp，连接上Linux服务器，然后将我们准备好的mysql的安装包传到服务器中。
  2. 对mysql进行解压，命令是 tar xvf 文件名
  3. 安装numactl（必要组件，不安装会导致后面的步骤出现依赖的问题。）
sh
yum -y install numactl
1

3.1. 卸载mariadb（这是系统自带的数据库，不卸载会导致MySQL安装失败。）

sh
rpm -qa | grep -i mariadb
rpm -e --nodeps mariadb-libs-5.5.60-1.el7_5.x86_64（这个文件名字是上一步查出来的）
1
2

3.2. 安装mysql，按顺序安装下面4个rpm（版本号可能不一样）。

sh
rpm -ivh mysql-community-common-5.7.23-1.el7.x86_64.rpm
rpm -ivh mysql-community-libs-5.7.23-1.el7.x86_64.rpm
rpm -ivh mysql-community-client-5.7.23-1.el7.x86_64.rpm
rpm -ivh mysql-community-server-5.7.23-1.el7.x86_64.rpm
1
2
3
4
  1. 等待安装结束后，启动数据库。命令：
sh
systemctl start  mysqld.service
1
  1. 检查数据库是否运行成功，命令：
sh
systemctl status mysqld.service
1
  1. 数据库安装成功后，先生成一个默认密码，查看密码的命令：
sh
cat /var/log/mysqld.log | grep password
1
  1. 使用上一步获取的密码连接数据库，mysql -u root -p 这个步骤你们肯定没问题的。
  2. 进入数据库后，必须修改密码才能做其他的操作，所以修改密码为1qaz!QAZ，命令：
sql
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '1qaz!QAZ';
1

9、然后退出数据库，用新密码重新登录。 10. 然后创建一个具有远程访问权限的账号。有3条语句，如下：

sql
create user 'root'@'%' identified with mysql_native_password by '1qaz!QAZ';  
grant all privileges on *.* to 'root'@'%' with grant option;
flush privileges;
1
2
3

10.1. 为了让数据库的密码能修改为123456，所以我们需要对数据库进行一些配置。命令如下，完成后，就可以通过navicat来随意修改密码了。

sql
# 查看数据库的密码规则
SHOW VARIABLES LIKE 'validate_password%';   
# 修改密码强度要求
set global validate_password_policy=LOW;   
# 修改密码长度要求
set global validate_password_length=6;
1
2

  1. 好了数据库的设置结束了，你现在可以尝试能不能用navicat来连接了。
  2. 如果不能，那八成是端口的问题。所以检查阿里云的控制台的安全组是否开放端口。
  3. 通过命令查看当前已经开放的端口：
sh
netstat -ntlp
1
  1. 如果不存在3306，那么通过以下2个命令打开3306端口号。将3306端口添加到防火墙例外并重启：
sh
firewall-cmd --zone=public --add-port=3306/tcp --permanent
firewall-cmd --reload
